IoT Smart Home Project

MallaReddy University - Department of Computer Science & Engineering

Overview

The IoT Smart Home project is an innovative home automation system developed by final-year students of Malla Reddy University. It integrates IoT technology and Raspberry Pi to create a smart, responsive, and energy-efficient environment. Designed for ease of use, scalability, and real-time control, the system automates lighting, temperature, and appliance control based on sensor data and user preferences.

Objective

The primary aim of the project is to create a connected home ecosystem that ensures comfort, security, and optimized energy usage. The system uses motion sensors, temperature sensors, and smart switches controlled through a centralized Raspberry Pi hub. Users can control devices remotely via a web interface or mobile app.

Features

  • Automated lighting control based on occupancy
  • Temperature regulation using smart thermostats
  • Remote access via smartphone or web dashboard
  • Real-time monitoring of electricity usage
  • Fire and gas leak detection with emergency alerts
  • Voice assistant integration (Alexa/Google Assistant)

Technologies Used

  • Raspberry Pi 4 as the main controller
  • Python for backend automation scripts
  • MQTT protocol for sensor-device communication
  • Node-RED for event-driven programming
  • HTML, CSS, and JavaScript for the control dashboard
  • IoT sensors: PIR, DHT11, Flame Sensor, Gas Sensor

Benefits

  • Reduces energy bills by up to 30%
  • Enhances home security and surveillance
  • Improves convenience through automation
  • Accessible to elderly and differently-abled individuals
  • Customizable and expandable to future IoT devices

Use Cases

  • Automatically turning off lights in unoccupied rooms
  • Adjusting AC based on room temperature and time of day
  • Sending alerts to user phones during gas leaks or fire
  • Logging power usage trends for each device
  • Using voice commands to control appliances

Future Scope

Future enhancements include AI-based behavior prediction, solar energy integration, biometric access, and geofencing. The team also plans to deploy this project in smart campus hostels to improve energy efficiency and enhance student living experience.
